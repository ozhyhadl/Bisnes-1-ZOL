# Content SEO Workflow

## Purpose
Define the step-by-step process for creating, optimizing, and publishing SEO-friendly content.

## Workflow Stages

### 1. Content Planning
- [ ] Identify target keyword/topic
- [ ] Research search intent (informational, navigational, transactional)
- [ ] Check competition for target keyword
- [ ] Define target audience for the content
- [ ] Plan content structure (headings, sections)

### 2. Content Creation
- [ ] Write compelling, unique title (includes target keyword)
- [ ] Write meta description (150–160 chars, includes keyword)
- [ ] Structure content with proper heading hierarchy (H1 → H2 → H3)
- [ ] Include target keyword naturally in first 100 words
- [ ] Write at least 300 words of meaningful content
- [ ] Add relevant internal links (2–5 per page)
- [ ] Include images with descriptive alt text
- [ ] Ensure content matches search intent

### 3. Technical SEO Check
- [ ] Page has unique title tag
- [ ] Page has unique meta description
- [ ] Page has proper canonical URL
- [ ] Page has correct indexing directive
- [ ] Open Graph tags are set
- [ ] Twitter Card tags are set
- [ ] Structured data is valid (if applicable)
- [ ] URL is clean and descriptive

### 4. Quality Check
- [ ] No duplicate content with other pages
- [ ] No spelling or grammar errors
- [ ] All links are working
- [ ] Images are optimized (WebP, lazy loading)
- [ ] Page loads within performance budget
- [ ] Mobile responsive

### 5. Publishing
- [ ] Add page to navigation (if applicable)
- [ ] Update sitemap
- [ ] Verify page is not blocked by robots.txt
- [ ] Submit page to Google Search Console (optional)
- [ ] Monitor indexing status

### 6. Post-Publish Review (Week 1)
- [ ] Page is indexed in Google
- [ ] No crawl errors reported
- [ ] Analytics tracking is working
- [ ] Check initial performance metrics

## Content Types & SEO Requirements

| Content Type | Min Length | Structured Data | Internal Links | Images |
|---|---|---|---|---|
| Landing Page | 500 words | Organization | 3–5 | Yes |
| Blog Post | 800 words | Article | 3–7 | Yes |
| Product Page | 300 words | Product | 2–4 | Yes |
| FAQ Page | As needed | FAQPage | 2–3 | Optional |
| About Page | 300 words | Organization | 2–3 | Yes |

## Anti-Patterns
- ❌ Publishing pages without meta description
- ❌ Keyword stuffing (unnatural repetition)
- ❌ Thin content (under 100 words with no value)
- ❌ Duplicate content across pages
- ❌ Missing alt text on images
- ❌ Publishing without checking mobile layout
